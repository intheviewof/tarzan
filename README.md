# tarzan
> [!WARNING]
> tarzan is still in progress, and an infant project  

a custom uBlock Origin list

## what is tarzan?
tarzan is a list of custom filters i have created for my own browsing experience. it doesn't add much over other filters

## how do i install tarzan?
in uBlock Origin settings -> filter lists, add the following urls as filters:

- adblock: `https://raw.githubusercontent.com/intheviewof/tarzan/refs/heads/main/tarzan_adblock.txt`

### which filters should i use?
if you just want a simple adblocking experience, only the adblock list should be enough
